# MythiLens – AI‑Powered Cultural Heritage Platform

MythiLens is a web-based application built using the Base44 platform and powered by Google Gemini AI.  
It enables users to explore cultural heritage through AI-based monument analysis and ancient text translation.

## Features
- Heritage Scanner (Image-based monument analysis)
- Ancient Text Translator (Text/Image/File input)
- AI-powered cultural & historical insights
- History dashboard for saved analyses

## Tech Stack
- Frontend: React, TypeScript, Tailwind CSS
- Backend: Base44 (Backend-as-a-Service)
- AI Model: Google Gemini AI (via InvokeLLM integration)

## Architecture
- Frontend communicates with Base44 SDK
- Base44 handles authentication, database, and file storage
- Google Gemini AI provides AI intelligence

## Note
Due to Base44 free trial limitations, direct export is not available.
This repository contains architecture, documentation, and sample code structure.
